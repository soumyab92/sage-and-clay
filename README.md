# SAGE & CLAY — Clinical Skincare with Sensorial Warmth

SAGE & CLAY is a modern, high-end e-commerce experience showcasing dermatologist-tested skincare routines that balance active clinical ingredients with botanical soothing elements.

## 🌿 Brand Philosophy
Clinical skincare does not need to feel cold or sterile. SAGE & CLAY pairs pure, proven actives (like Vitamin C, Retinol, Niacinamide, and Ceramides) with tactile, comforting botanical bases (like Colloidal Oatmeal, Sage, Tea Tree, and Cucumber) to nurture skin and support a strong lipid barrier.

---

## ✨ Features

### 🛒 E-Commerce SPA Architecture
* **Dynamic Product Catalog:** Filter by category (Cleansers, Serums, Moisturizers, Sunscreens, Masks, Eye Care) or by concerns (Acne, Anti-Aging, Hydration, Brightening, Sensitivity).
* **Detailed Product Pages (PDP):** Image gallery with hover-to-zoom magnification, standard vs value size variant selector, real ingredient glossaries, and dermatologist cautions.
* **Responsive sliding drawers:** Live sliding drawers for the Cart and Wishlist access.
* **Wishlist Gifting Feature:** Allows users to share their wishlist directly via generated copyable sharing links.
* **Return/Exchange request flow:** Simulated return lookup and pre-paid USPS label authorization.

### 🎨 Visual & Motion Enhancements
* **Brand Colors & Typography:** Elegant Cormorant Garamond serif headings and Inter sans-serif details on a warm sand background.
* **Scroll-shrinking header:** Header scales down on scroll to optimize reading views.
* **Parallax image depth:** Subtle 3D movement when scrolling past hero illustrations.
* **Auto-rotating announcement bar:** Slides through multiple promotional highlights.
* **Clinical stats counters:** Count-up animation for trial statistics on viewport scroll.
* **Mobile navigation drawer:** Optimized menus and item carousels for touch device widths.

## 🚀 Running Locally

The site runs on pure HTML5, CSS3, and JavaScript, packaged as a lightweight Single Page Application (SPA).

1. Open `index.html` directly in any web browser, OR:
2. Serve via a local web server (e.g. `npx live-server` or `python -m http.server 8000`).
